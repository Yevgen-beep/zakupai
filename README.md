# ZakupAI

MVP-платформа для автоматизации госзакупок РК: поиск и разбор лотов, быстрые TL;DR, риск-скоринг, финкалькулятор (НДС/маржа/пени), генерация писем/жалоб, интеграции с n8n/Flowise и API сервисы на FastAPI.
